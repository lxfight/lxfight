| 项目名称                                                                                         | 描述                                                                                              |
| :----------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------ |
| 🧠 [**astrbot_plugin_mnemosyne**](https://github.com/lxfight/astrbot_plugin_mnemosyne)           | 一个为 AstrBot 实现的基于 RAG 技术的长期记忆插件，让机器人能“记住”更久远的对话。                  |
